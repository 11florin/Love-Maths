# 💙 Love Maths – Interactive Maths Game

Love Maths is an educational and interactive web application designed to help users practise basic arithmetic in a fun and engaging way.  
The game offers four types of maths challenges: **addition, subtraction, multiplication, and division**, all generated randomly. Users receive instant feedback and can track their progress through a built‑in scoring system.

This project was created as part of my studies with **Code Institute**, where I am currently a student on the Full Stack Software Development programme.

---

## 🎮 Features

- Randomly generated maths questions  
- Four game modes:
  - ➕ Addition  
  - ➖ Subtraction  
  - ✖️ Multiplication  
  - ➗ Division  
- Automatic answer validation  
- Score tracking for correct and incorrect answers  
- Fully responsive layout for mobile, tablet, and desktop  
- Clean, modern interface with colour‑coded buttons and Font Awesome icons

## 🚀 How to Play

1. Open the webpage in your browser.  
2. Select a maths operation using one of the coloured buttons.  
3. Enter your answer in the input field.  
4. Press **Submit Answer** or hit **Enter**.  
5. Check your score and keep practising.
